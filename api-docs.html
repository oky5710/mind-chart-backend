HRV 분석

Heart Rate Variability
POST /hrv
HRV 분석 결과 저장
tpLog / vlfLog / lfLog / hfLog 는 수신하지 않음 — 원 점수만 저장
Request Body
{
  "examinedAt":  "2024-06-01T09:00:00Z",
  "hospital":    "서울병원",     // optional
  "memo":        "피로감 있음",  // optional
  // Time Domain
  "mhr":         72,
  "sdnn":        45.2,
  "rmssd":       38.1,
  "psi":         82.5,
  // Frequency Domain (원 점수만)
  "tp":          1240.5,
  "vlf":         320.1,
  "lf":          480.2,
  "hf":          440.2,
  "lfNorm":      52.1,
  "hfNorm":      47.9,
  "lfHfRatio":   1.09,
  "ectopicBeat": 2,
  "srd":         18.3,
  "result":      "정상"
}
Response 201
{
  "id":          1,
  "userId":      "clx...",
  "examinedAt":  "2024-06-01T09:00:00Z",
  "hospital":    "서울병원",
  "memo":        "피로감 있음",
  "mhr":         72,
  "sdnn":        45.2,
  "rmssd":       38.1,
  "psi":         82.5,
  "tp":          1240.5,
  "vlf":         320.1,
  "lf":          480.2,
  "hf":          440.2,
  "lfNorm":      52.1,
  "hfNorm":      47.9,
  "lfHfRatio":   1.09,
  "ectopicBeat": 2,
  "srd":         18.3,
  "result":      "정상",
  "createdAt":   "2024-06-01T09:00:00Z"
}
GET /hrv
전체 HRV 목록 조회 — 최신순 정렬
Response 200
[{ "id": 2, "examinedAt": "...", ... }, { "id": 1, ... }]
GET /hrv/:id
HRV 단건 조회
Response 200
{ "id": 1, "examinedAt": "2024-06-01T09:00:00Z", ... }
PATCH /hrv/:id
HRV 수정 — 모든 필드 부분 수정 가능
Request Body (예시)
{
  "memo":   "수정된 메모",
  "result": "경계"
}
Response 200
{ "id": 1, "memo": "수정된 메모", ... }
DELETE /hrv/:id
HRV 삭제
Response 200
{ "id": 1, "examinedAt": "2024-06-01T09:00:00Z", ... }
