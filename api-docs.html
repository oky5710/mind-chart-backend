이벤트 (Events)

Method Endpoint 설명 Request Body / Query Response
POST /events 이벤트 등록 
{
  "date": "2024-06-01",
  "type": "WORK",
  "title": "발표 준비",
  "description": "중요 발표",  // optional
  "sentiment": "NEGATIVE",
  "intensity": 4
}

// type: WORK | PERSONAL |
//        MEDICAL | SOCIAL | OTHER
// sentiment: POSITIVE | NEGATIVE | NEUTRAL
// intensity: 1 ~ 5
 
{
  "id": "clx...",
  "date": "2024-06-01T00:00:00.000Z",
  "type": "WORK",
  "title": "발표 준비",
  "sentiment": "NEGATIVE",
  "intensity": 4,
  ...
}

(최신순 정렬)		 없음 
[
  { "id": "clx...", "date": "...", ... },
  ...
]
GET /events/:id 이벤트 단건 조회 없음

  }

{
  "medicationId": "clx...",
  "date": "2024-06-01",

  "reason": "부작용 감량"  // optional
}

// changeType: START | STOP |
//   DOSAGE_CHANGE | PAUSE | RESUME
 
{
  "id": "clx...",
  "changeType": "DOSAGE_CHANGE",
  "previousDosage": 1,
  "newDosage": 0.5,
  ...
}
GET /medications/changes 복약 변경 기록 조회
(최신순 정렬)		 없음 
[
  {
    "id": "clx...",

    ...
  }
]
